Mobile Services
     We can bring the library to you!  If you are a Secaucus resident and are regularly
     unable to visit the library due to age, disability or serious illness, we can arrange a
     delivery of library items.  For more information or set up a delivery, please call
     (201) 330-2083.

Computer Passes
     The library welcomes both Secaucus and non-Secaucus residents to the library.
     If you are not a Hudson County resident or BCCLS library card holder, you may
     purchase a computer pass for $1.00. Each pass allows you to use our computers
    for up to two hours.

Notary Public
     Notary services are available for Secaucus residents and employees.  Please note      that effective January 1, 2012, you will be charged $1.00 for each page requiring a     notary signature after the first initial page. Notaries are available Monday to Friday,     however it is recommended to call before visiting the library.
